Overview

Thinking about a new career in software development? Start here! In this one-day workshop, you'll get a taste of a day in the life of a software developer.

Get a sneak peek of the Code Fellows learning experience, find out how websites are built, and code a webpage yourself using industry-standard tools and professional coding practices.

Outcomes

At the end of this workshop, you will:

  • Understand the basics of what it means to be a professional software developer.
  • Understand how HTML, CSS, and JavaScript work together to create the modern web.
  • Create a complete website.
  • Know how to deploy your website and be able to share your accomplishments with friends and family.
  • Know what whiteboarding is and complete a whiteboarding activity in collaboration with your classmates.
  • Be able to identify whether a career in coding and the Code Fellows program is right for you.

Prerequisites

Set up Sublime Text Editor, Terminal, and Git. Follow the guide for your operating system below.

  • Mac OS X guide
  • Ubuntu guide
  • Windows guide

Having trouble following these guides? An hour before the start of the workshop will be dedicated to helping students get their computers set up with these tools.

Computer Requirements

Students are required to bring their own laptop. Before the workshop, please download:

  • The latest version of Google Chrome
  • A text editor, such as Sublime Text or Atom

Code of Conduct Policy

Punctuality, participation in discussions, completion of assignments, and demonstration of professional courtesy to others are required. Attendance will be taken at the beginning of the workshop. Passing requires attendance to the full workshop.
